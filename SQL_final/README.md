# Анализ базы данных 
 Компания купила крупный сервис для чтения книг по подписке.

## Данные

- о книгах;
- об издательствах;
- об авторах;
- пользовательские обзоры книг.

## Задача

Проанализировать базу данных на языке запросов SQL и сформулировать ценностное предложение для нового продукта.

## Используемые библиотеки

pandas, sqlalchemy
